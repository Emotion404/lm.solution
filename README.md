<b>二、项目努力目标</b>

　　1.完备的 Java 知识点，即拿即用！

　　2.完备的 Method 封装，即拷即用！

　　3.完备的 Spring MVC/Boot/Cloud 框架，开箱即用！

　　4.让老猿省时，菜鸟不费力！

<b>三、特点</b>

　　1.项目开发运行完全在Linux物理机上，目前环境为 DeepIn(GNU/Linux) 系统

　　2.注释完备，资料齐全
